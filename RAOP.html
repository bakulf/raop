Abstract

The purpose of this project is an ontology that allows students and researchers to map and annotate linguistics and stylistic aspects of lyric text as well as a prose. It aims to create an ontology based on W3C and Semantic web that shapes classical and modern rhetorical studies able to link to from DublinCore alla TEI/ CIDOC-CRM. The annotation of rhetorical is therefore important not only for the linguistic point of view, but also for discovering different styles of writing, purpose and effect of written documents, and for better natural language understanding in general. This ontology could be expanded to every standard of Semantic web, for instance RDF, RDFs e OWL.
